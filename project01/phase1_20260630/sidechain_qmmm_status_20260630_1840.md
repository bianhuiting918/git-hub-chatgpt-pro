# Project 01 Phase1 status: sidechain completion, strict geometry, QMMM gate

## Evaluated universe

- Input set: 5 rank-1 representatives, one per 90/80/70/60/50 sequence-similarity bin.
- PLACER sampling: 3 conformers per representative, 15 total conformers.
- Full ligand mapping: 15/15 mapped to full-length protein with full `bu2` ligand, no minimization.
- Sidechain completion: 15/15 completed with PDBFixer heavy-atom sidechain addition, no minimization.
- Fixed-pocket sidechain graft: 15/15 generated by local N-CA-C rigid transform from the reference fixed active/contact residues, no minimization.

## Tool-native and project filters

- PLACER produced output for all 15 conformers in this pilot set.
- PDBFixer completed 15/15 sidechain-heavy structures.
- Strict all-atom pocket geometry filter evaluated 15/15 sidechain-completed structures.
- Strict all-atom pocket geometry filter evaluated 15/15 fixed-pocket-grafted structures.

Project strict thresholds are downstream filters, not PLACER-native pass/fail labels:

- catalytic heavy RMSD max: 0.5 A
- fixed-pocket heavy RMSD max: 0.75 A
- ligand heavy RMSD max: 0.5 A
- key-distance absolute delta max: 0.35 A
