# Motif templates

This directory should contain layered Folddisco motif YAML files.

Current design principle:

```text
L0 = catalytic core
L1 = catalytic environment
L2 = substrate pocket
L3 = processing / interface / dynamics context
```

Do not encode every residue as a strict required motif point. Catalytic-core residues belong in L0. Pocket, processing, interface, and dynamics residues belong in L2/L3 and should usually contribute weighted evidence rather than hard exclusion.

Expected example files to be generated by Codex:

```text
petase_cutinase.layered.example.yaml
nylc_ntn_hydrolase.layered.example.yaml
```

See `../MOTIF_STRATEGY.md` for the full specification.
