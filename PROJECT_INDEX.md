# Enzyme reaction learning project index

Created: 2026-06-28  
Updated: 2026-06-28

This repository contains two related but separate research projects. They should be developed in sequence and kept as separate code paths until a shared utility package is clearly needed.

The updated research design is:

```text
Project 01: true-TS teacher
  specific enzyme/family + GS + true/refined TS + QM/MM or DFT barrier labels
  → mechanism-aware barrier scorer

Project 02: predicted-TS student
  enzyme + GS + reaction prior + predicted TS geometry/embedding + PLACER screening
  → catalytic-class-conditioned barrier proxy / catalytic potential ranking
```

## Vocabulary

- **GS（ground state，基态）**: the bound reactant/substrate complex.
- **true TS（真实过渡态）**: a transition-state structure or TS-like state that is constructed, optimized, refined, or validated by QM/MM, DFT, reaction-coordinate scans, constrained optimization, or a consistent TS template.
- **predicted TS（预测过渡态）**: a TS geometry, TS embedding, TS-like conformer, or reaction-state prior generated by a TS prediction model; it is not treated as quantum ground truth.
- **barrier proxy（势垒 proxy）**: `ΔG‡`, `ΔE_TS-GS`, `ΔΔG‡`, `ΔΔE‡`, or a learned score calibrated to the QM/MM or DFT protocol.
- **catalytic potential（催化潜力）**: predicted ability of an enzyme environment to stabilize the relevant TS relative to GS. It is not the same as experimental `kcat`, because experimental activity can also depend on substrate access, product release, stability, expression, and assay conditions.

## Project 01 — Enzyme-specific true-TS barrier scorer

Path: `projects/01-specialized-ts-aware-scorer/`

Purpose: build a high-confidence **true-TS teacher model** for a specific enzyme, enzyme family, or well-defined catalytic class where GS and true/refined TS states are available or can be generated consistently.

First-round enzyme systems:

- serine hydrolases（丝氨酸水解酶）
- PETase / cutinase-like PET hydrolases（PET 水解酶 / 角质酶类似体系）
- CALB / lipase / esterase benchmark systems（CALB / 脂肪酶 / 酯酶基准体系）

Later or exploratory systems:

- PUase / polyurethane hydrolase（聚氨酯水解酶）
- nylonase（尼龙酶）
- metal hydrolases, redox enzymes, laccase, carbonic anhydrase（后期跨催化类型测试）

Main question:

> Given a specific enzyme/family with known GS and true/refined TS states, can a model learn the QM/MM or DFT-defined TS-GS barrier and rank new mutants or generated designs?

Key boundary:

- Project 01 **may use true TS as an input**.
- Project 01 is not primarily trained on experimental `kcat` or `kcat/KM`.
- The primary labels are computed barriers and relative computed barriers: `ΔG‡`, `ΔE_TS-GS`, `ΔΔG‡`, `ΔΔE‡`, and within-template ranking.
- PLACER is used as a conformer generation/screening interface, but raw PLACER structures are not treated as final quantum labels.

## Project 02 — Predicted-TS catalytic-class barrier predictor

Path: `projects/02-general-enzyme-prediction/`

Purpose: build a **predicted-TS student model** that approximates Project 01 behavior without requiring a true TS at inference time.

Main question:

> If Project 01 learns TS-GS barrier patterns for multiple enzymes and catalytic categories, can Project 02 use a TS prediction model plus PLACER screening to predict catalytic potential within a target catalytic type when true TS coordinates are unavailable?

Key boundary:

- Project 02 **must not require true TS at inference**.
- It may use Project 01 true-TS labels, teacher predictions, true-TS embeddings, and QM/MM barriers during training or distillation.
- At inference, it should use GS, protein/ligand context, reaction prior, and predicted TS geometry/embedding.
- The first-stage target is computed barrier proxy / catalytic potential ranking, not direct experimental `kcat` prediction.

## Relationship between the two projects

```text
Project 01: true-TS teacher
  protein + GS + true/refined TS + QM/MM/DFT labels
  → ΔG‡ / ΔΔG‡ / ranking / mechanism components

        ↓ distillation, alignment, upper-bound comparison

Project 02: predicted-TS student
  protein + GS + reaction prior + TS prediction model output + PLACER screening
  → ΔG‡ proxy / catalytic potential ranking / mechanism components
```

Project 01 answers:

> With a true TS available, can the model learn how the enzyme environment stabilizes TS relative to GS?

Project 02 answers:

> Without a true TS available at inference, can predicted TS priors close the performance gap between a GS-only model and the true-TS teacher?

## Shared principle

Do not train a black-box `embedding → activity` model as the primary claim. The central claim is:

> Enzyme catalytic potential can be modeled by decomposing computed TS-GS barrier changes into reaction-state geometry, electrostatic TS stabilization, conformational ensemble accessibility, and energetic penalties. Project 01 learns this decomposition with true TS supervision; Project 02 tests whether predicted TS priors can transfer it to practical inference settings.
