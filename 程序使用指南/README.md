# 程序使用指南

这个目录用于集中保存可复用的项目运行手册，目标是让新的 Codex 窗口或其他协作者可以快速接手已有计算流程。

## 当前指南

- [gmx-cp2k指南](./gmx-cp2k指南/README.md)
  - GMX-CP2K 调用方式
  - PETase WT 与突变体无水 QMMM 总势能计算
  - 批量队列、并发控制、监控和结果汇总

## 使用原则

1. 先读对应子目录的 `README.md`，确认服务器、目录、软件环境和计算口径。
2. 运行前先检查旧进程和已有结果，避免重复提交长任务。
3. 修改并发数前先看 `free -h`、`uptime` 和现有 `gmx_mpi_d` 进程 RSS。
4. 长计算必须保留日志、`.exit`、`.done` 和结果 CSV，便于断点续跑。
5. 不把密码、token、私钥写入仓库。
