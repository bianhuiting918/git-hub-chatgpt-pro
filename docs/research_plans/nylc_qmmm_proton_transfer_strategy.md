# NylC 催化 PA66-L2 第一化学事件的 GROMACS–CP2K QM/MM 研究计划

**版本：v2.0**  
**更新时间：2026-07-24**  
**当前阶段：正在探索第一化学事件，尚未确定质子先转移、亲核攻击先发生，还是二者协同发生。**

---

## 1. 研究问题

本项目不再把第一步预先定义为“Thr267 质子转移”，而是把目标定义为：

> 确定成熟 NylC 催化 PA66-L2 酰胺键水解时的第一化学事件、质子受体、反应顺序和自由能势垒。

需要区分以下三种基本情况：

1. **质子转移优先**：Thr267-OγH 先去质子化，形成可辨认的 Thr267-Oγ⁻ 中间体，随后亲核攻击底物羰基。
2. **协同过程**：Thr267-Oγ 对羰基碳的攻击与质子转移在同一个过渡区内耦合，没有稳定的独立 Thr267-Oγ⁻ 中间体。
3. **亲核攻击优先**：先发生明显的 Oγ–C 成键和羰基四面体化，之后再完成质子转移。

最终不能只回答“质子去了哪里”，还必须回答：

- 质子转移和 Oγ–C 成键的先后顺序；
- 是否存在稳定中间体；
- 哪条路径具有最低自由能势垒；
- Asp306、Asp308、Thr267 的 N 端氨基、Lys189、水和离去酰胺氮分别承担什么作用。

---

## 2. 必须先明确的体系定义

### 2.1 底物名称不能只写“L2”

本计划暂沿用项目中的“PA66-L2”命名。正式计算前必须固定以下信息，并写入单独的底物说明文件：

- 完整二维化学结构；
- 单体重复单元的定义；
- 两个末端的封端方式；
- 总电荷；
- 被切割酰胺键；
- 羰基碳、羰基氧、离去氮和 Thr267-Hγ 的统一原子编号；
- 是否实际研究的是 PA6,6 模型低聚物，还是 6-aminohexanoate/Ahx 低聚物。

不同“L2”定义对应不同的末端电荷、底物构象和离去基团碱性，不能混用。

### 2.2 研究成熟酶的底物水解，不与自剪切混为一谈

NylC 前体在 Asn266/Thr267 之间自剪切，成熟后 Thr267 成为新的 N 端亲核残基。本项目研究的是：

```text
成熟 NylC + PA66-L2 → 酰胺键水解
```

而不是：

```text
NylC 前体 → Asn266/Thr267 自剪切
```

2023 年 NylC 论文对自剪切和底物水解提出了相关但不完全相同的催化模型。自剪切中提出的 Lys189/水网络不能未经验证直接移植到 PA66-L2 水解。

### 2.3 保留完整寡聚体环境

推荐使用已经平衡好的 NylC 四聚体作为 MM 环境，只把一个活性中心及其底物设为 QM 区。原因是：

- 活性口袋和入口可能受到相邻亚基约束；
- 截断成单体可能改变水通道和静电环境；
- 邻近亚基残基即使不进入 QM 区，也应保留在 MM 环境中。

第一阶段只研究一个活性中心，不要同时在四个活性中心放置反应底物。

## 3. 已知实验与结构先验

Negoro 等在 2023 年的晶体结构、突变和经典 MD 研究中提出：

- Thr267 是成熟酶的 N 端亲核残基；
- Asp308–Asp306–Thr267 构成关键催化网络；
- Lys189、Asn219 和 Tyr146 对 nylon 底物水解具有重要作用；
- Tyr146 和 Asn219 可能参与稳定四面体氧负离子；
- D306A、D308A 和 T267A 会严重破坏催化相关过程。

但这些结果没有直接给出 PA66-L2 水解的 QM/MM 自由能面，也没有证明第一步质子一定由某个固定残基接受。因此，以上内容应作为**优先假设**，而不是计算结论。

## 4. 需要比较的竞争机制

### H1：Thr267 N 端氨基参与的经典 Ntn 路径

```text
Thr267-OγH → Thr267-Nα
                 ↓
Thr267-Oγ 进攻底物羰基
                 ↓
Nα 再向离去酰胺氮供质子
```

需要判断：

- OγH→Nα 是否先形成稳定离子对；
- 还是 Oγ–C 成键时才发生质子迁移；
- Nα 是否确实参与后续离去基团质子化。

### H2：Asp308–Asp306–Nα–Thr267 网络调控路径

这是 NylC 结构论文对底物水解提出的优先模型。Asp308 和 Asp306可能通过氢键及静电作用调节 Thr267-Nα/Oγ 的酸碱性，而不一定直接接收 Thr267-Hγ。

需要区分：

- **直接质子受体**：质子真正停留在 Asp306 或 Asp308 上；
- **间接催化**：Asp306/Asp308 不接收质子，只改变 Nα/Oγ 的 pKa、氢键和反应几何。

### H3：Asp306 直接接受 Thr267-Hγ

```text
Thr267-OγH → Asp306-Oδ
```

该路径必须单独测试，但不能仅凭短氢键就判定成立。只有在自由能面上形成可重复的 Asp306-COOH 状态，并且该状态连接真实反应路径，才能称为直接质子接受。

### H4：Lys189/水介导质子中继

可能形式包括：

```text
Thr267-OγH → 水 → Lys189
```

或：

```text
Thr267-OγH → 水链 → 离去酰胺氮
```

这里必须明确 Lys189 的质子化状态：

- Lys189-NH3+ 不能再作为普通质子受体；
- 只有中性 Lys189-NH2 才能直接接收质子；
- 中性 Lys189 是否稳定，需要独立验证，不能因为文献提出其可能是一般碱就直接固定为中性。

### H5：直接或近直接的离去氮质子化

在某些 Ntn 水解酶中，亲核攻击与离去酰胺氮质子化高度耦合。NylC 中可能出现：

```text
Oγ–C 成键 + H 向离去氮迁移
```

也可能通过 Thr267-Nα 完成中继。该路径需要与 H1–H4 在相同 QM 区和相同理论水平下比较。

## 5. 起始构象选择

### 5.1 不使用单一 Michaelis complex

从已完成的经典 MD 或约束释放 MD 中选择至少 3–5 个相互独立的反应前构象。不能只选择 Oγ–C 距离最短的一帧。

建议筛选指标：

- Thr267-Oγ 到被切割羰基碳的距离；
- Oγ–C–O 亲核攻击角；
- 羰基氧与 Tyr146/Asn219 的氢键；
- 被切割 C–N 键的取向；
- Thr267-Hγ 与 Nα、Asp306、Asp308、水和离去氮的距离；
- 活性位点水网络是否稳定；
- 底物在口袋中的 register 是否一致。

推荐挑选：

1. 一个攻击几何最理想的构象；
2. 一个水可进入的构象；
3. 一个水相对缺失但蛋白氢键网络稳定的构象；
4. 如资源允许，再加入两个不同底物构象。

### 5.2 每个候选机制至少从两个独立构象开始

如果某条路径只在一个特殊快照上出现，而在其他快照中不存在，不能直接判定为主机制。

## 6. 质子化微状态设计

第一步反应高度依赖 Thr267、Asp306、Asp308 和 Lys189 的初始质子化状态。建议先测试以下有限集合，而不是穷举所有组合。

### S0：结构论文优先状态之一

```text
Thr267-NαH2 / OγH
Asp306-COOH
Asp308-COO−
Lys189-NH3+
```

### S1：Asp 质子位置交换

```text
Thr267-NαH2 / OγH
Asp306-COO−
Asp308-COOH
Lys189-NH3+
```

### S2：中性 Lys189 假设

```text
Thr267-NαH2 / OγH
Asp306-COOH
Asp308-COO−
Lys189-NH2
```

仅在短 QM/MM MD 中该状态稳定、且没有立即从环境获得质子时继续使用。

### S3：预形成的 Thr 两性离子状态

```text
Thr267-NαH3+ / Oγ−
```

用于判断“先去质子化”是否形成真实自由能最低点，而不是直接把它当作反应物。

### 微状态筛选规则

对每个状态进行：

1. QM/MM 能量最小化；
2. 2–5 ps 无偏 QM/MM MD；
3. 检查质子是否立即返回、转移或导致结构崩溃；
4. 检查总 QM 电荷和自旋设置；
5. 比较多个快照上的相对稳定性。

如果一个微状态只能靠强约束维持，就不应作为独立反应物最低点。

## 7. QM 区域设计

### 7.1 原则

任何可能发生质子转移、成键或断键的原子都必须进入 QM 区。MM 原子不能可靠描述质子化状态改变。

为公平比较竞争机制，优先使用统一的大 QM 区，而不是每条路径使用不同 QM 区。

### 7.2 推荐核心 QM 区

至少包括：

- Thr267：N 端氨基、完整侧链和必要的主链片段；
- Asp306 侧链；
- Asp308 侧链；
- PA66-L2 被切割酰胺键；
- 被切割键两侧至少各保留一个完整化学单元；
- 与 Thr267-Hγ 或反应中心直接形成稳定氢键的 1–3 个水分子。

### 7.3 推荐扩展 QM 区

加入：

- Lys189 侧链；
- Asn219 侧链；
- Tyr146 中参与氧负离子稳定的主链/侧链片段；
- 直接参与质子中继的水；
- 其他与羰基氧直接氢键的基团。

如果要测试 Lys189 直接接收质子，Lys189 必须进入 QM 区。如果要测试水桥，整个质子线上的水必须进入 QM 区。

### 7.4 边界处理

- 尽量在普通 Cα–Cβ 单键处切分侧链；
- 不要切在被反应酰胺键、羧酸共轭体系或质子中继链内部；
- Thr267 的 Nα 和 Oγ 必须处于同一 QM 电子结构中；
- 对 Thr267 主链边界至少测试两种合理切法；
- 记录所有 link atom 的 QM_INDEX、MM_INDEX 和边界图。

### 7.5 QM 区收敛测试

至少比较：

```text
QM-small：核心催化基团
QM-large：加入 Lys189、Asn219、Tyr146 和直接水网络
```

在同一批 ES、受限反应构象和过渡区构象上比较相对能量与关键力。如果扩大 QM 区后机制发生改变或势垒变化明显，使用 QM-large 作为生产设置。

## 8. GROMACS–CP2K 技术路线

当前仓库记录的环境为：

- GROMACS 2023.1；
- CP2K 2024.1；
- double precision；
- GROMACS 通过 libcp2k 调用 CP2K；
- electrostatic embedding/GEEP 类型耦合。

### 8.1 使用自定义 CP2K 输入

推荐：

```text
qmmm-cp2k-qmmethod = INPUT
```

通过 `gmx grompp -qmi` 提供自定义 CP2K 输入。CP2K 输入必须满足接口要求，包括 `RUN_TYPE ENERGY_FORCE`、QM 总电荷、多重度和坐标设置。

### 8.2 初始电子结构水平

建议将以下设置作为起点，而不是未经测试的最终答案：

- PBE-D3(BJ)；
- DZVP-MOLOPT-SR-GTH 或经验证的 DZVP-MOLOPT-GTH；
- GTH-PBE 赝势；
- GPW/Quickstep；
- 闭壳层 singlet，除非体系定义显示其他自旋态。

可用 BLYP-D3 作为方法敏感性对照。代表性 ES、过渡区和中间体结构可进一步做更高水平单点校正。

### 8.3 网格收敛

不能机械照搬一个 CUTOFF。建议在至少两个结构上测试：

```text
CUTOFF：400 / 500 / 600 Ry
REL_CUTOFF：50 / 60 / 70 Ry
```

至少包含：

- 一个平衡 ES 构象；
- 一个 Oγ–C 已明显缩短、电子密度变化较大的反应构象。

选择使相对能量和关键反应原子受力稳定的最低成本组合。

### 8.4 SCF 稳定性

CP2K 2024.1 在 SCF 不收敛时默认终止。生产计算不得使用 `IGNORE_CONVERGENCE_FAILURE` 掩盖问题。

建议：

- `EPS_SCF` 从 1×10⁻⁶ Ha 开始；
- 增加合理的 `MAX_SCF`；
- 使用稳定的 OT/预条件设置；
- 每步复用前一步波函数；
- 在反应区电荷重排明显时检查 SCF 迭代数是否突增；
- 单独保存出现 SCF 困难的窗口并重新优化设置。

### 8.5 系综与时间步长

- 经典 MM 阶段可先在 NPT 下平衡密度；
- 正式 GROMACS–CP2K QM/MM 生产阶段使用 NVT；
- 不建议直接用该接口做 NPT 反应自由能，因为 QM/MM 力对 virial/压力的贡献存在接口限制；
- 初始时间步长使用 0.5 fs；
- 如质子振动、能量漂移或 SCF 不稳定明显，降至 0.25 fs；
- 不对 QM 区内涉及质子转移的 X–H 键施加 LINCS/固定键约束。

### 8.6 旧版本接口的重启测试

当前 GROMACS 版本早于后续针对 QM 子区居中和 checkpoint 重启的改进。正式大规模运行前必须做一次短重启测试：

1. 连续运行一段轨迹；
2. 从 checkpoint 重启；
3. 比较重启前后 QM 区坐标、能量和力是否连续；
4. 检查周期性边界下 QM 区是否被错误拆分。

## 9. 第一阶段：无偏 QM/MM MD

### 9.1 目的

不是指望无偏轨迹自然跨越化学势垒，而是：

- 检查微状态稳定性；
- 确认候选质子受体；
- 识别稳定水网络；
- 给后续增强采样提供起点；
- 排除明显不合理的初始模型。

### 9.2 推荐规模

DFT 级 QM/MM 下，原先建议的单条 100–500 ps 轨迹成本过高，也没有必要作为第一步。

推荐初筛：

```text
3–5 个起始快照
×
每个微状态 2–5 ps 预平衡
+
5–10 ps 生产轨迹
```

只对最稳定的 1–2 个微状态延长到 20–30 ps。

### 9.3 分析指标

持续统计：

- d(Oγ–Hγ)；
- d(Hγ–Nα)；
- d(Hγ–Oδ-Asp306)；
- d(Hγ–Oδ-Asp308)；
- d(Hγ–Owater)；
- d(Hγ–Nleaving)；
- d(Oγ–Ccarbonyl)；
- Oγ–C–O 攻击角；
- d(C–Nleaving)；
- 羰基氧与 Tyr146/Asn219 的氢键；
- 水分子驻留时间和交换；
- Thr267-Nα 与 Asp306/Asp308 的氢键。

输出距离分布、角度分布、氢键占据率和代表构象，不只报告平均值。

## 10. 第二阶段：第一化学事件的探索性扫描

### 10.1 先扫描亲核攻击坐标，但不能据此单独下结论

定义：

```text
q_attack = d(Thr267-Oγ, Ccarbonyl)
```

从约 3.2–3.5 Å 缩短到约 1.4–1.6 Å，进行约束最小化或短约束 QM/MM MD。

该扫描回答：

- Oγ 接近羰基时，Hγ 是否自然转向某个受体；
- 羰基是否形成四面体化；
- 哪些氢键网络随攻击建立。

但必须明确：

> 如果在一维 q_attack 扫描中质子没有自动转移，不能据此排除质子转移路径，因为质子转移可能存在与 q_attack 正交的隐藏势垒。

因此一维 Oγ–C 扫描只用于生成候选路径，不能作为最终机制证据。

### 10.2 为每个候选受体定义质子转移差分坐标

对受体 A：

```text
q_PT(A) = d(Oγ,Hγ) - d(Hγ,A)
```

受体 A 依次测试：

- Thr267-Nα；
- Asp306-Oδ；
- Asp308-Oδ；
- 稳定催化水的 O；
- 中性 Lys189-Nε；
- 离去酰胺 N。

差分坐标比单独拉短 H–A 更适合描述原 O–H 断裂与新 H–A 成键。

### 10.3 水桥路径的特殊处理

如果水分子在轨迹中频繁交换，不能把某一个水编号永久指定为“催化水”。

处理顺序：

1. 先在短轨迹中寻找驻留时间长的水；
2. 用固定水做初步路径；
3. 再检查其他水交换是否改变机制；
4. 必要时使用 coordination number 或 path collective variable，而不是固定原子编号。

## 11. 第三阶段：二维自由能面

### 11.1 核心二维坐标

第一张关键自由能面应为：

```text
CV1 = q_attack
CV2 = q_PT(A)
```

每个主要受体分别构建二维自由能面。优先级建议：

1. Thr267-Nα；
2. Asp306；
3. 水/Lys189 网络；
4. 离去酰胺 N；
5. Asp308 直接接受。

### 11.2 为什么必须做二维

二维自由能面可以区分：

- PT-first：先沿 q_PT 进入稳定中间体，再缩短 q_attack；
- attack-first：先缩短 q_attack，再发生质子转移；
- concerted：最低自由能路径沿对角方向跨越单一鞍区；
- 两条机制并存：出现两个可比较通道。

### 11.3 采样方式

核心方案不依赖 PLUMED，可使用 GROMACS pull/umbrella sampling 或 AWH。只有确认当前 GROMACS 构建已经启用并验证 PLUMED 后，才使用 path CV、string 或 metadynamics。

推荐分两级：

#### 粗筛网格

- q_attack 间隔约 0.2–0.25 Å；
- q_PT 间隔约 0.2–0.25 Å；
- 每窗口先短平衡，再采样 0.5–1 ps；
- 目的：定位最低自由能通道和需要加密的区域。

#### 精细网格

只在最低自由能路径附近加密：

- 间隔约 0.1 Å；
- 每窗口采样 3–10 ps，具体由自相关时间决定；
- 过渡区、浅中间体和水重排区增加采样。

不要一开始铺满整个高分辨率二维网格，否则成本会失控。

### 11.4 自由能重建

可采用：

- WHAM；
- MBAR；
- AWH 原生自由能估计。

必须检查：

- 相邻窗口重叠；
- 前后半段自由能曲线；
- block analysis；
- 正向和反向初始化是否存在滞后；
- 不同起始快照是否给出相同通道。

## 12. 后续成键断键坐标

第一化学事件明确后，再加入：

```text
q_CN = d(Ccarbonyl, Nleaving)
```

必要时定义成键/断键组合坐标：

```text
q_acyl = d(C,Nleaving) - d(C,Oγ)
```

如果 Nα 先接收 Hγ，再向离去氮供质子，则至少需要两个质子坐标：

```text
q_PT1 = OγH → Nα
q_PT2 = NαH → Nleaving
```

这种多步中继不适合硬塞进一个简单距离。优先使用：

- 分段二维自由能面；
- path collective variable；
- string method；
- 或经过验证的多维 metadynamics。

## 13. 机制判定标准

### 13.1 “先质子转移”成立的标准

必须同时满足：

- 二维自由能面上存在独立的质子转移后局部最低点；
- 此时 Oγ–C 尚未明显成键；
- 解除约束后该状态能稳定存在，而不是立即返回；
- 多个起始快照均能得到相同状态。

### 13.2 “协同机制”成立的标准

- 最低自由能路径中 q_attack 和 q_PT 同时显著变化；
- 只出现一个主要自由能鞍区；
- 找不到稳定的 ThrO⁻ 或独立质子转移中间体。

### 13.3 “Asp306 接受质子”成立的标准

- Hγ 完整转移到 Asp306-Oδ；
- Asp306-COOH 是可重复的局部最低点；
- 该状态沿最低自由能路径继续通向四面体中间体或酰基酶；
- 不是单纯的短 O–H···O 氢键。

### 13.4 “Asp306/Asp308 仅间接催化”的标准

- 两个 Asp 不形成稳定接质子状态；
- 但删除其 QM 描述、改变质子化状态或做电荷中和对照会明显改变：
  - Thr267 Oγ/Nα 的电子状态；
  - 亲核攻击几何；
  - 过渡区自由能；
  - 水可进入性。

### 13.5 “水桥机制”成立的标准

- 质子沿连续水网络转移；
- 水网络在反应窗口中具有足够占据率；
- 更换初始水编号后仍得到相同拓扑路径；
- 自由能低于或接近蛋白直接中继路径。

## 14. 过渡区验证

GROMACS–CP2K 接口更适合 QM/MM 动力学和自由能采样，而不是把 IRC 作为唯一验证手段。

推荐验证顺序：

1. 从二维自由能面的鞍区提取多个构象；
2. 解除偏置，分别向前和向后进行短 QM/MM 轨迹；
3. 使用不同随机速度进行 10–30 次短轨迹；
4. 理想过渡区构象应接近一半回到反应物、一半进入产物/中间体；
5. 如资源允许，再用 CP2K 独立 QM/MM、NEB 或静态 TS 方法进行辅助确认。

因此，本项目的主要证据链应为：

```text
二维自由能面
+
最低自由能路径
+
无偏下坡轨迹/committor
+
多快照重复
```

而不是只依赖一次静态 TS 优化。

## 15. 方法学敏感性与误差控制

### 15.1 必做对照

- 至少 3 个独立 Michaelis complex；
- 至少 2 个合理质子化微状态；
- QM-small 与 QM-large；
- PBE-D3 与一个替代泛函/代表性高水平单点；
- 有水和无固定催化水的起始构象；
- 正向和反向初始化窗口。

### 15.2 收敛目标

建议将以下条件作为“可下机制结论”的门槛：

- 相邻窗口存在充分重叠；
- 延长后势垒变化不超过约 1 kcal/mol；
- 不同快照的主路径一致；
- 不同快照势垒差异可解释并报告；
- QM 区扩大后不改变质子受体身份；
- 代表性更高水平单点不颠倒路径排序。

如果不同合理设置给出不同机制，应把结论表述为构象或微状态依赖，而不是强行选唯一机制。

```text

第一势垒为：ΔG‡ = ______ ± ______ kcal/mol。
```
